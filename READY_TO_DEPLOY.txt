                      SIMPLE 3-STEP DEPLOYMENT
================================================================================

STEP 1: GITHUB (2 minutes)
─────────────────────────────────────────────────────────────────────────────
  $ cd nagabay
  $ git init
  $ git add .
  $ git commit -m "Initial commit: Nagabay Smart Health Navigator"
  $ git remote add origin https://github.com/YOUR_USERNAME/nagabay.git
  $ git branch -M main
  $ git push -u origin main

STEP 2: VERCEL (1 minute)
─────────────────────────────────────────────────────────────────────────────
  1. Go to https://vercel.com
  2. Click "New Project"
  3. Import your nagabay repository
  4. Add Environment Variable:
     Name: GOOGLE_GENERATIVE_AI_API_KEY
     Value: YOUR_API_KEY_HERE
     Environments: Check all three
  5. Click "Deploy"

STEP 3: VERIFY (2 minutes)
─────────────────────────────────────────────────────────────────────────────
  Browser Console:
  $ fetch('/api/health').then(r => r.json()).then(d => console.log(d))
  
  Expected: { success: true, status: "healthy" }

  Then test the full API:
  $ fetch('/api/triage', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        age: 35,
        barangay: "Abella",
        symptoms: ["fever", "cough"],
        duration: "3 days",
        severity: "moderate"
      })
    }).then(r => r.json()).then(d => console.log(d))

  Expected: Triage analysis with triageLevel, recommendedFacility, etc.

================================================================================
                      TROUBLESHOOTING QUICK FIX
================================================================================

Problem: "API service encountered an error"
Solution:
  1. Verify API key is in Vercel Environment Variables
  2. Variable name MUST be: GOOGLE_GENERATIVE_AI_API_KEY
  3. Redeploy: Deployments → Latest → ... → Redeploy
  4. Wait 2-3 minutes
  5. Hard refresh: Ctrl+Shift+R (or Cmd+Shift+R on Mac)

Problem: Health check returns 503
Solution:
  1. Check API key is valid at https://aistudio.google.com/
  2. Check API quotas (you may need to upgrade)
  3. View logs: Vercel Dashboard → Deployments → Functions → /api/triage

Problem: Still not working?
Solution:
  1. Check Vercel logs in detail
  2. Read DEPLOY_WITH_VERCEL_GATEWAY.md for troubleshooting section
  3. Contact Vercel support: https://vercel.com/support
